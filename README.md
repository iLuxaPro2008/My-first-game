# Стрельба по шарикам

## Описание

Игра для тренировки внимательности и скорости реакции. За 60 секунд нужно кликать по зелёным шарикам (+2 очка) и избегать красных (-5 очков). Доступна настройка размера поля (от 2×2 до 7×7) и уровня сложности (лёгкий, средний, сложный).

## Системные требования

- ОС: Windows 7/8/10/11
- Процессор: от 1 ГГц
- ОЗУ: от 512 МБ
- .NET Framework 4.5 или выше

## Установка

1. Скачайте файл `BurstBallsSetup.exe`
2. Запустите установщик
3. Следуйте инструкциям на экране
4. Запустите игру через ярлык на рабочем столе

## Использование

1. Выберите размер поля (Ширина и Высота) → нажмите «Применить»
2. Выберите уровень сложности: «Лёгкий», «Средний» или «Сложный»
3. Нажмите «Старт»
4. Кликайте по зелёным шарикам, избегайте красных
5. Через 60 секунд игра покажет ваш счёт
